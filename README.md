## Run Locally

Clone repository

```bash
  git clone https://github.com/FabianEscarate/letterSoup
```

Go to the project directory

```bash
  cd letterSoup
```

use current node version

```bash
  nvm use
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
